# Lua基本構文マニュアル

本マニュアルは、スクリプト言語「Lua」の基本構文を網羅した実用リファレンスです。埋め込み型言語としての特性を活かした、軽量で高速なプログラミングのための必須知識をまとめています。

---

## 1. 基本仕様とコメント

### 1.1 大文字・小文字の区別

Luaは**大文字と小文字を厳密に区別**します。たとえば、`Value`、`value`、`VALUE` はすべて異なる変数として扱われます。

### 1.2 文の区切り

文末のセミコロン（`;`）は**任意**です。改行があれば自動的に文の区切りとみなされますが、1行に複数の文を書く場合はセミコロンで区切ります。

### 1.3 コメントの書き方

コード内にメモを残すためのコメント構文です。

```lua
-- 1行コメント：行末までがコメントになります

--[[
    複数行コメント：
    ここからここまでが
    すべてコメントとして扱われます
--]]

```

## 2. 変数とデータ型

### 2.1 スコープ（ローカルとグローバル）

Luaの変数はデフォルトで**グローバル変数**になります。予期せぬバグを防ぐため、原則として `local` キーワードを付与して**ローカル変数**として定義することが推奨されます。

```lua
global_var = "どこからでもアクセス可能"
local local_var = "現在のブロック内でのみ有効"

```

### 2.2 主要なデータ型

Luaは動的型付け言語です。変数自体ではなく、値そのものが型を持っています。

| 型名 (type) | 説明 | 例 |
| --- | --- | --- |
| `nil` | 値が存在しないことを示す（未定義の変数もこれ） | `nil` |
| `boolean` | 真偽値 | `true`, `false` |
| `number` | 数値（内部的には主に倍精度浮動小数点数） | `42`, `3.14` |
| `string` | 文字列 | `"Hello"`, `'World'` |
| `function` | 関数（Luaでは関数もファーストクラスオブジェクト） | `function() ... end` |
| `table` | 唯一の複合データ構造（配列・連想配列として使用） | `{}` |

## 3. 演算子

### 3.1 算術演算子

基本的な四則演算に加え、剰余やべき乗が用意されています。

* `+` （加算）
* `-` （減算）
* `*` （乗算）
* `/` （除算）
* `%` （剰余）
* `^` （べき乗）

### 3.2 比較演算子

比較の結果は `true` または `false` で返されます。「等しくない」の表記に **`~=`** を使用する点に注意してください。

* `==` （等しい）
* `~=` （等しくない）
* `<` （より小さい）
* `>` （より大きい）
* `<=` （以下）
* `>=` （以上）

### 3.3 論理演算子

* `and` （論理積）：両方が真なら真
* `or`  （論理和）：いずれかが真なら真
* `not` （論理否定）：真偽値を反転

> **注意：偽（False）とみなされる値**
> Luaにおいて、条件式で「偽」と判定されるのは **`false` と `nil` のみ**です。数値の `0` や空文字列 `""` は「真（True）」として扱われます。

### 3.4 文字列結合

文字列の結合には **`..`**（ドット2つ）を使用します。

```lua
local message = "Hello, " .. "World!"  -- "Hello, World!"

```

## 4. 制御構造（条件分岐・ループ）

すべての制御構造は、最後に **`end`** で閉じる必要があります。

### 4.1 条件分岐（if文）

`if` の後には `then` が必須です。

```lua
local score = 85

if score >= 90 then
    print("優秀")
elseif score >= 70 then
    print("合格")
else
    print("不合格")
end

```

### 4.2 ループ処理

#### 数値指定の for 文

`for 変数 = 初期値, 最大値, [増減値] do` の形式で記述します。増減値を省略した場合は `1` ずつ増加します。

```lua
-- 1から5まで1ずつ増加してループ
for i = 1, 5 do
    print(i)
end

-- 10から2まで2ずつ減少してループ
for i = 10, 2, -2 do
    print(i)
end

```

#### while 文 / repeat 文

条件が満たされている間（または満たされるまで）繰り返します。

```lua
-- while文：最初から条件を判定
local count = 1
while count <= 3 do
    print(count)
    count = count + 1
end

-- repeat文：末尾で条件を判定（最低1回は実行される）
local num = 1
repeat
    print(num)
    num = num + 1
until num > 3

```

## 5. テーブル（Table）

Luaにおける唯一のデータ構造であり、配列、辞書（連想配列）、オブジェクトの役割をすべて担います。

### 5.1 配列としての利用

Luaの配列は、原則として**インデックスが `1` から始まる**のが大きな特徴です。

```lua
local fruits = {"apple", "banana", "orange"}

print(fruits[1]) -- "apple" （0ではなく1から始まる）

-- テーブルの要素数は「#」演算子で取得可能
print(#fruits)   -- 3

```

### 5.2 連想配列（キー・値ペア）としての利用

文字列などをキーとして値を格納できます。

```lua
local user = {
    name = "田中",
    age = 30
}

-- アクセス方法は2通り
print(user["name"]) -- "田中"
print(user.age)     -- 30 （ドット記法も可能）

```

### 5.3 テーブルの反復処理（ipairs と pairs）

テーブルの全要素をループ処理する場合、用途に応じてイテレータ関数を使い分けます。

```lua
-- ipairs：配列用（インデックス1から順に連続するデータのみ処理）
local list = {"A", "B", "C"}
for index, value in ipairs(list) do
    print(index, value)
end

-- pairs：連想配列・順不同用（すべてのキー・値ペアを処理）
local dict = {x = 10, y = 20}
for key, val in pairs(dict) do
    print(key, val)
end

```

## 6. 関数（Function）

### 6.1 基本定義と呼び出し

引数を受け取り、処理結果を返す基本的な構造です。

```lua
local function add(a, b)
    return a + b
end

local result = add(5, 3) -- 8

```

### 6.2 複数戻り値（マルチリターン）

Luaの関数は、一度に**複数の値を返す**ことができます。

```lua
local function get_coordinate()
    local x = 100
    local y = 200
    return x, y  -- 複数の値を返す
end

local posX, posY = get_coordinate()
print(posX, posY) -- 100  200

```
